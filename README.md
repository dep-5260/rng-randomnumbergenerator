# rng-randomnumbergenerator

### Usage
```js
const rng = require('rng-randomnumbergenerator')

let Number1 = rng.generateRandomNumber(100)

let Number2 = rng.generateRandomNumberLimit(1, 150)

console.log(Number1)
console.log(Number2)
})
```
